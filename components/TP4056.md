# TP4056 Battery Charger + DC-DC Boost Converter (HW-357 / J5019)

## Overview
Combined TP4056 linear charger and adjustable boost converter module for single-cell Li-ion/LiPo batteries. Handles charging from solar panel and provides regulated output to power the system.

## Key Specs
| Spec | Value |
|------|-------|
| **Charger IC** | TP4056 (CC/CV linear charger) |
| **Charge Current** | Up to 1A (default, set by Rprog ≈1.2kΩ) |
| **Cut-off Voltage** | 4.2V ±1% |
| **Battery Type** | 1-cell Li-ion / LiPo (3.7V nominal / 4.2V full) |
| **Boost Output** | Adjustable 4.3V – 27V (via trim pot) |
| **Boost Max Current** | ~1.4A @ 5V, ~0.8A @ 9V, ~0.6A @ 12V |
| **Input Voltage** | DC 4.5V – 8V (USB or solar panel) |
| **Quiescent Current** | ~0.5 mA (idle) |
| **PCB Size** | ~33 × 24 × 6 mm |
| **Weight** | ~11 g |

## Wiring (Power Chain)
```
Solar Panel (5V) ──→ B+ / B- pads on TP4056 (battery charging)
                          │
                     [18650 Battery]
                          │
               OUT+ / OUT- ──→ 5V boosted output
                                   │
                          ESP32 VIN (5V pin)
                          Servo VCC (5V)
                          Other 5V components
```

| Terminal | Connection |
|----------|-----------|
| IN+ / IN- | Solar panel positive / negative (or USB 5V) |
| B+ / B- | 18650 battery positive / negative |
| OUT+ / OUT- | Boosted output → ESP32 VIN, Servo VCC, etc. |

## LED Indicators
| LED | State |
|-----|-------|
| Red | Charging |
| Blue/Green | Battery fully charged |

## Protections
- Over-current protection (boost side)
- Short-circuit protection (boost side)
- Thermal protection (boost side)
- **No low-voltage battery protection** — remove battery if voltage drops too low

## Important Notes
- **Not connected to ESP32 GPIOs** — purely power management.
- Set boost output to ~5V using the trim pot (adjust while measuring with multimeter).
- Avoid heavy loads during charging — may prevent proper charge termination.
- If solar panel output is weak, charging current will be lower than 1A.
- Suitable for solar-powered portable projects like this Sun Tracker.

## Suggested Output Voltage
Set trim pot to **~5V** output:
- ESP32 VIN accepts 5–12V (has onboard 3.3V regulator)
- Servo MG996R operates at 4.8–6V (5V is ideal)
- Both can share the same 5V rail from OUT+
